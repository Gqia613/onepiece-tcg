# 青クザン（OP12-040）完全ガイド ― OP16/ST33環境・フラッグシップ準拠構築

## はじめに：このガイドの方針

本ガイドは、記事の丸写しではなく **ゲームの構造原理から1枚ずつ根拠づけて** 青クザンを構築する。
情報源は日本語のみ・**2026年4月以降＝OP16/ST33/EB04 の現行環境**に限定した。
青クザンはOP12で登場したリーダーだが、**ST-33「スタートデッキ 青 クザン」**（スモーカー/コビー/サカズキ/ボルサリーノ/ガープ）と**EB04**（イッショウ/アイスタイム）で大幅に強化され、現行環境の形は初期（2025年6月フラッグシップ優勝時）とは別物になっている。本構築はこの**現行プールに最適化**している。

> 参照：メルカード「青クザン OP16.5 Ver1.0」/ note・cardrush 各フラッグシップ入賞レシピ（下部リンク）を**参考**にしつつ、カード効果は当リポジトリの `official-full.json`（公式スナップショット）で全数照合した。

| 項目 | 値 |

| ライフ | 5 |
| パワー | 5000 |
| 効果 | **自分の特徴《海軍》を持つカードの効果で自分の手札からカードが捨てられた時、捨てた枚数分カードを引く。** |

### このリーダーの本質＝「捨てる＝実質引く」への変換器

OPCGの根本原理「**ライフ＝手札／手札を削り切ることが真の勝利条件**」に照らすと、青クザンの強さは明快だ。

- 通常、カードの効果で「手札を1枚捨てる」＝**コスト（−1枚）**。
- クザン下では、海軍カードの効果ディスカードが即座に**1ドロー**を生む → **捨てても手札が減らない**。
- つまり「手札1枚を捨てる：〇〇する」という**追加効果を、手札を減らさずに毎回撃てる**。ディスカードを要求するカードのコストが実質ゼロになる。

これにより、①デッキ下送り除去、②アタック不可（フリーズ）、③相手ハンデス、④ブロッカー展開――を**リソースを吐かずに連打**し、相手の手札を枯らして勝つ「青単コントロール／ミッドレンジ」が成立する。ライフ5＝**受けの土台**も厚い。

## Step 1：このデッキの強み・弱み（構造分析）

### 強み

1. **リソース無限機関**：たしぎ/センゴク/コビーで毎ターン手札を掘り、ディスカードが引き直しに化ける。長期戦でほぼ息切れしない。
2. **「除去せず無力化」できる**：デッキ下送り（スモーカー/サカズキKO）と**フリーズ（アイスタイム/6クザン）**を併用。フリーズは"除去"ではないため、**効果耐性・KO時効果・「KOされない」を貫通**して相手のアタッカーを黙らせられる。
3. **防御が厚い**：2000カウンター多数＋アイス塊暴雉嘴（+4000カウンター）＋ブロッカー（コビー/ボルサリーノ/アラマキ）＋ライフ5。アグロを構造的に受け切る。
4. **手札破壊プラン**：イッショウ/サカズキで相手の手札6枚以上を狙い撃ち。カウンター資源そのものを奪う。

### 弱み

1. **リーサル速度が遅い**：縛って勝つデッキで、決定的なワンショット火力に乏しい。**長期戦で"より太いリソース"に負ける**（→紫エネル）。
2. **ディスカード燃料が前提**：捨てる弾（手札）が尽きると機関が止まる。**初手の噛み合い**に依存。
3. **明確なフィニッシャー不足**：大型は8000どまり。10000超の耐性持ちを能動的に処理しづらい（フリーズで時間を稼ぐしかない）。
4. **盤面除去が"デッキ下/バウンス"中心**＝**リムーブが一時的**（デッキに戻る）。再展開されるとイタチごっこになりうる。

### 構築の思考プロセス

- **コスト帯配分**：1c=4（たしぎ）/ 2c=8 / 3c=8 / 4c=4 / 5c=8 / 6c=12 / 8c=4 ＋ イベント（1c×3・5c×3）。
  6cが12枚と多いが、**ボルサリーノは実質3c**で着地するため体感カーブは滑らか。序盤は2〜3cの機関で受けつつ掘り、5c以降で盤面を制圧する典型的な青コントロールのカーブ。
- **ディスカード燃料の供給と需要のバランス**：燃料を"要求"するカード（スモーカー/センゴク/クザン/サカズキ/イッショウ/ガープ056/ヒナ/アイス塊/アイスタイム）が多い一方、燃料を"供給"するのはドロー機関そのもの。**捨てる→引く→また捨てる**の循環が回るよう、サーチ（たしぎ/センゴク）を最優先で厚くした。
- **カウンター密度**：2000カウンター＝スモーカー/ヒナ/ボルサリーノ（各4）＝12枚、1000＝たしぎ/コビー/センゴク/サカズキ/ガープ＝20枚、+4000イベント3枚。**受けは環境屈指の厚さ**でライフ5と噛み合う。
- **除去の質の使い分け**：小型はスモーカー/アラマキで**デッキ下**（KO時効果を踏まない）、大型はクザン/アイスタイムで**フリーズ**（耐性貫通）。KOに頼らないのが青の思想。

## Step 5：立ち回りガイド

### 序盤（ターン1〜2）
- **たしぎ**を最優先で置き、足りない海軍（センゴク/スモーカー）をサーチ。ブロッカーとしても機能。
- 後手なら**コビー**でブロッカーを立てつつ手札を掘る（原理：後手は手札プラン＝受けて増やす）。
- 相手が並べてきたら**スモーカー**で2体まとめてデッキ下。対アグロはここで盤面差を作る。

### 中盤（ターン3〜5）
- **センゴク**を毎ターン回して機関を起動。海軍2枚サーチ＋捨て札の還元で、手札が減らないまま盤面に触り続ける。
- **サカズキ(ST33)**でアタックしつつ相手をハンデス。KO時効果で海軍を呼び戻し、盤面が途切れない。
- 相手の大型が立ったら**6クザン**で殴りながら、その大型を凍結。除去できない耐性持ちもこれで沈黙。
- 相手の手札が6枚以上に膨らんだら**イッショウ**で2枚デッキ下＝**カウンター資源を直接破壊**。

### 終盤（ターン6〜）
- **ガープ(ST33/056)**で2体同時展開し、盤面を一気に厚くしてライフを詰める。
- 詰めの局面では**ヒナ**で相手ブロッカーを無力化してリーサルを通す。
- 相手の反撃アタッカーは**アイスタイム**で2体凍結＝1ターン完全に攻撃を止めて、こちらのクロックだけ進める。
- **アラマキ**を蓋に置き、毎ターンの除去＋8000ブロッカーで相手の巻き返しを封じる（原理：手札を枯らした相手はブロッカー貫通も除去も返せない）。

## マリガン基準

### 必ずキープ
- **たしぎ**：初動＝機関の起点。1枚は欲しい。
- **センゴク**：最強の掘削。あれば手札は自然に整う。

### 状況キープ
- **スモーカー**：先攻でも後攻でも腐らない。対アグロ想定ならキープ優先。
- **コビー**：後手で受けたいときキープ。ブロッカー＋掘削。
- 3cまでのカードが2枚以上あるなら重い札（イッショウ/クザン/ガープ）は1枚まで抱えてよい。

### 返す
- **アラマキ/ガープ(056)/アイスタイム**が初手に固まっている場合＝動けないので返す。
- 6cクザン単騎など**重い札だけの手**は積極的に切る。

### 理想の初手（例）
`たしぎ ＋ センゴク ＋ スモーカー ＋ コビー ＋ アイス塊暴雉嘴`
→ 1〜3ターンで機関が回り、除去・ブロッカー・カウンターが揃った鉄板の受け型。

## デッキの弱点と苦手マッチ

### 構造的弱点
- **リーサルが遅い**：縛って勝つ設計ゆえ、ワンショット火力がない。**ゲームを長引かせるほど"より太いリソース"に押し切られる**リスク。
- **燃料切れ**：ディスカード前提。掘削が引けないと機関が止まり、ただの中型ビートに劣化する。
- **一時的リムーブ**：デッキ下/バウンス/フリーズはいずれも"戻る/解ける"。再展開されると処理し切れないことがある。

### 苦手マッチアップ（現OP16環境）
- **紫エネル（TIER1・コントロール）＝不利寄り**：10000＋効果耐性の神を能動除去できず、ライフ回復＋大型連打のリソース戦で先に息切れしがち。**回答はフリーズ（アイスタイム/クザン）で時間を稼ぎ、イッショウのハンデスで神官供給を細らせる**こと。長期戦は覚悟。
- **有利寄り**：**赤系アグロ**（スモーカー＋カウンター＋ブロッカーで受け切る）、**黒黄ティーチ**（デッキ下送りでKO時効果を踏まない・ハンデス合戦でこちらの機関が上回る）、**青緑ルフィ等の展開ミッド**（1体ずつ処理＋フリーズ）。

### 注意すべきプレイングミス
- **捨て札の選択ミス**：ディスカードは"引き直せる"とはいえ、後で使う海軍やカウンターを切らないよう、余剰の重複札から捨てる。
- **手札を6枚未満に保つ判断**：相手のサカズキ/イッショウのハンデス条件は「相手手札6枚以上」。膨らませすぎない（クザン自身も手札5枚以上でコスト+1になる点に注意）。
- **フリーズの撃ちどころ**：アイスタイムは"次の相手エンドまで"。**相手の攻撃ターン直前**に撃つのが最大効率（先撃ちは解けて無駄になる）。

## 環境に応じた調整枠

| 入替候補 | 差し替え先 | 理由 |
|---|---|---|
| OP12-044 サカズキ（7c・2ドロー＋ドン付与） | アラマキ or ガープ056 と1枚 | ドローとドン加速を足したい／コントロールミラーを重く見るとき |
| OP06-051 つる（5c・バウンス除去） | ヒナ 1〜2枚 | "戻す"除去を増やし、耐性キャラを能動処理したいとき |
| OP12-046 ゼファー（5c・手札2捨て＋バウンス起動） | イベント枠 | 燃料供給と手札に戻す除去を厚くしたいとき |
| アイスタイム 3→2 | ヒナ or スモーカー +1 | アグロが増えたら受け寄せ、コントロールが増えたら凍結寄せ |

**調整方針**：アグロ多数の環境 → スモーカー/コビー/アイス塊を厚く（受け）。コントロール多数 → アイスタイム/イッショウ/OP12-044サカズキを厚く（凍結＋手札破壊＋リソース勝負）。

## 結論

青クザンは「**手札を捨てる＝実質引く**」という一点で、青の除去（デッキ下・バウンス・**フリーズ**）と手札破壊を**ノーコストで連打**するコントロール。ライフ5と厚いカウンターで受け、相手の手札を枯らして勝つ、OPCGの根本原理に忠実なデッキだ。**ST33/EB04で受け・除去・展開が全て底上げされた現行環境版**は、アグロと展開ミッドに強く、明確な弱点は紫エネルのようなリソースお化けとの長期戦。フリーズと手札破壊でその弱点を最小化するのがプレイの肝となる。
