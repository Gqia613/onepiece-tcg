# デッキ別 立ち回り・勝ち筋リファレンス（AI戦略反映用）

各デッキの「強い動き」を日本語の権威ある解説（note PROS/tcg-portal/cardrush/メルカード等・CLAUDE.md許可ソース）から調査し、
**AIの盤面評価(`evalState`)へ反映できる指標**に分解したもの。`src/70-ai.js` の per-leader プロファイル(§B-3)の根拠。
※記事は丸写しせず要点を抽出。効果値は公式カードリストで別途検証する。

凡例（AI指標の語彙）= 盤面パワー / キャラ数 / 手札 / ライフ / 利用可能ドン(active) / ドン総数(ramp) / デッキ枚数(longevity) / ブロッカー数 / 脅威の質(高パワー・効果耐性) / 相手キャラ数(横展開) / 相手アタック数(連撃耐性)。

---

## enel（紫エネル）— style: コントロール / ランプ（★AI探索が最も苦手）
- **勝ち筋**: ドン6固定の独自システム。第2ターン以降「毎ターン無料で4ドン付与」の緩急で、除去耐性の10000(エネル本体)を毎ターン展開し、神官の連撃＋ドン-イベント(放電/雷獣/神の裁き)で盤面と相手リソースを制圧。
- **テンポ**: 序盤はアグロで相手の手札を削る → 中盤以降は10000連打＋イベントでコントロール。コスト7以上不可（6ドン内で完結）。
- **主要資源/エンジン**: ドン循環(active/rested/付与)、10000効果耐性キャラ、0コストのドン-イベント。
- **AI指標**: ドン総数(ramp)・利用可能ドン=**強加点**（受け/イベント/付与の燃料）、脅威の質(10000耐性)=加点、短期の盤面パワーは**相対的に抑制**（ドン使い切りアグロを是正）。→ 実測でpuctがドンを使い切る(残0.27 vs heuristic 0.84)弊害の直接対策。

## lucy（赤青ルーシー）— style: コントロール / 除去
- **勝ち筋**: イベント/ステージを「実質カウンター」に変換。赤の火力除去＋青のバウンスで盤面の取り合いをリードし、サボ(7c/9000ブロッカー)で蓋。
- **テンポ**: 序盤はコンボパーツを手札に揃え「ドレスローザ王国」を設置 → 除去で制圧。**ライフで受けすぎない**（イベントをカウンター資源に温存）。
- **弱点**: 横に並べられると処理が追いつかない。
- **AI指標**: 手札(イベント=カウンター資源)=加点、ブロッカー=加点、**相手キャラ数(横展開)=マイナス**（処理しきれない盤面を嫌う）、ライフ温存より手札温存。

## ace（赤青エース）— style: ミッドレンジ
- **勝ち筋**: ライフ3・被弾するほどドローが進む。12000ブロッカー(ニューゲート)と速攻ロジャーで攻守両立する万能型。
- **テンポ**: ミッドレンジ。被弾を過度に恐れない（ドローに変換）。
- **AI指標**: バランス（ほぼ現状の重み）。ブロッカー=加点、自ライフが減ること(被弾)への減点を**緩和**（ドロー利得）。

## nami（青黄ナミ）— style: コントロール / 高耐久（受け）
- **勝ち筋**: 手札1枚でリーダーを7000化し序中盤のアタックを受け止める。ライフ移動で発動するドロー＋黄のライフ回復で見た目以上に高耐久。対アグロ(エネル小型連撃/ハンコック強襲)に強い。
- **テンポ**: 受け切ってからリソース勝ち。ロングゲーム。
- **AI指標**: 手札(7000pump/カウンター資源)=加点、ライフ=加点(回復)、ブロッカー=加点、デッキ枚数(longevity)=加点。**防御重視プロファイル**。

## hancock（青黄ハンコック）— style: コントロール / グラインド（ゾンビ）
- **勝ち筋**: トリガーでアタック回数+ライフバーン(攻)＆ドロー(防)の攻防一体。デッキトップ操作(ナミュール/カヤ)で8モリアをライフに仕込み**複数回ヒール**。先攻「3→5→7→9」の大型展開で処理不能盤面→回復で相手のライフ/リソースを削り切る。
- **テンポ**: ロングゲーム志向。
- **AI指標**: 盤面パワー(大型展開)=加点、ライフ=加点(複数回ヒール)、card advantage(ドロー)=加点、デッキ枚数(longevity)=加点。

## teach（黒黄ティーチ）— style: ミッドレンジ（環境TOP）
- **勝ち筋**: リーダー効果で相手アタックを「実質手札1枚で1回」受ける(防御)。KO時効果を持つ黒ひげ達をリーダー効果でアタックに引き込み、除去+ドローで盤面と手札を整える。**ドン10で「ゼハハ」=ティーチ展開＋相手ライフ奪取**のバースト。
- **テンポ**: ミッドレンジ。場持ちの良いキャラで攻撃先を誘導。
- **弱点**: 連続アタックに弱い（リーダーで吸えるのは1回/ターン）。
- **AI指標**: 盤面(場持ち)=加点、手札(リーダー受け+カウンター)=加点、除去=加点、**ドン10到達**を一種のマイルストーンとして加点、**相手のアタック数(連撃)=脅威としてマイナス**（連撃で手札を削られる弱点）。

## yamato（黒ヤマト）— style: アグロ / トラッシュ蘇生（★新規追加 Part A）
- **リーダー**: OP16-079 ヤマト（黒・ライフ5・パワー5000）。**トラッシュからワノ国キャラが登場した時その1枚に【速攻】**。
- **勝ち筋**: 豊富な資源・複数回アタック・場残りする高パワーで圧をかける。6ヤマト→トラッシュから8ヤマトを蘇生→速攻で即アタック等、アタック回数を稼いで攻め切る。
- **テンポ**: アグロ寄り（積極的に攻める）。
- **AI指標**: 盤面圧/アタック数=加点、トラッシュ資源(蘇生先)=加点、速攻=加点。**短期打点を素直に評価**（現状の評価とおおむね整合＝アグロは探索が得意）。

## まとめ表（per-leaderプロファイルの方向性）
| リーダー | 型 | 加点を強める指標 | 抑制/減点する指標 |
|---|---|---|---|
| enel | コントロール/ランプ | ドン総数・利用可能ドン・脅威の質 | 短期盤面パワー(使い切りアグロ) |
| lucy | コントロール/除去 | 手札(イベント)・ブロッカー | 相手キャラ数(横展開) |
| ace | ミッドレンジ | ブロッカー | 被弾減点を緩和 |
| nami | コントロール/受け | 手札・ライフ・ブロッカー・デッキ | （防御重視） |
| hancock | コントロール/グラインド | 盤面パワー・ライフ・ドロー・デッキ | — |
| teach | ミッドレンジ | 盤面・手札・除去・ドン10到達 | 相手アタック数(連撃)を脅威化 |
| yamato | アグロ/蘇生 | 盤面圧・アタック数・トラッシュ | （現状評価で概ねOK） |

※この方向性を `evalState` のper-leader重み(§B-3)に落とし、**各リーダーのミラー測定(puct vs heuristic)で改善/横ばいの時だけ採用**（退行は不採用）。定性→定量の翻訳なので、効くかは測定で判定する（CLAUDE.md「測定が正」）。
